ML MR Portal Mockup
===================

How to Run:
-----------
1. Simply double-click on file "ML BR Portal.html" to open it in your web browser (Chrome, Edge, Firefox, etc.).
2. No installation or server is required.

Notes:
------
- This mockup uses Vanilla HTML/CSS/JavaScript.
- It requires an internet connection to load the Icons (Phosphor Icons) and Fonts (Google Fonts Inter).
- The "Login" button and "Submit" actions are simulated for demonstration purposes.

Folder Structure:
-----------------
- ML BR Portal.html  : The main entry point (Open this file).
- css/               : Stylesheets.
- js/                : Application logic.
